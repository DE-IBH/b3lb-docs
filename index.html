BigBlueButton Load Balancer

B3LB is a open source BigBlueButton API load balancer similar to Scalelite. It is designed to work in large scale-out deployments with 100+ BigBlueButton nodes and high attendee join rates.

The project started at IBH IT-Service GmbH in the fall 2020 and has been published in February 2021 during the second COVID-19 lock down in Germany.
